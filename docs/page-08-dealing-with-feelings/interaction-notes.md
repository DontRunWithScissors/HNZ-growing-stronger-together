# Interaction notes – Page 8

## Interaction rules

- Each car is tappable
- On tap:
  - the car fills with a Health NZ pipeline colour
  - fade-in animation only (300–400ms)
- Tapping again removes the colour
- Multiple cars can be selected at the same time

## Movement

- Cars do NOT move
- No sound
- No timer
- No scoring

This activity is a digital colouring exercise,
not a racing game.
